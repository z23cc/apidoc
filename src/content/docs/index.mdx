## 什么是 Claude Code？

Claude Code 是 Anthropic 推出的官方命令行工具（CLI），让你可以直接在终端中与 Claude AI 进行交互式对话，完成各种软件工程任务。

## 支持的模型

| 模型 | 说明 |
|------|------|
| **Claude Opus 4.6** | 最强大的模型，适合复杂任务 |
| **Claude Sonnet 4.6** | 平衡性能和速度 |
| **Claude Haiku 4.5** | 最快速度，适合简单任务 |
