## 什么是 Claude Code？

Claude Code 是 Anthropic 推出的官方命令行工具（CLI），它让你可以直接在终端中与 Claude AI 进行交互式对话，完成各种软件工程任务。

## 核心特性

### 1. 终端原生体验
Claude Code 运行在你的终端中，直接访问你的项目文件和开发环境，无需在浏览器和编辑器之间切换。

### 2. 智能代码理解
- 自动分析项目结构
- 理解代码上下文和依赖关系
- 支持多种编程语言

### 3. 内置开发工具
- **文件操作**: 读取、编辑、创建文件
- **代码搜索**: Glob 模式匹配和内容搜索
- **Git 操作**: 提交、分支管理、PR 创建
- **命令执行**: 运行构建、测试等 shell 命令

### 4. 可扩展性
- **MCP 服务器**: 连接外部工具和数据源
- **自定义技能**: 通过 SkillKit 安装社区技能
- **Hooks**: 在工具执行前后运行自定义脚本

## 支持的模型

Claude Code 默认使用最新的 Claude 模型：
- **Claude Opus 4.6** — 最强大的模型，适合复杂任务
- **Claude Sonnet 4.6** — 平衡性能和速度
- **Claude Haiku 4.5** — 最快速度，适合简单任务
