## What is Claude Code?

Claude Code is Anthropic's official command-line interface (CLI) tool that lets you interact with Claude AI directly in your terminal for various software engineering tasks.

## Core Features

### 1. Terminal Native Experience
Claude Code runs in your terminal with direct access to your project files and development environment — no switching between browser and editor.

### 2. Intelligent Code Understanding
- Automatic project structure analysis
- Understanding of code context and dependencies
- Multi-language support

### 3. Built-in Developer Tools
- **File Operations**: Read, edit, and create files
- **Code Search**: Glob pattern matching and content search
- **Git Operations**: Commits, branch management, PR creation
- **Command Execution**: Run builds, tests, and shell commands

### 4. Extensibility
- **MCP Servers**: Connect external tools and data sources
- **Custom Skills**: Install community skills via SkillKit
- **Hooks**: Run custom scripts before/after tool execution

## Supported Models

Claude Code uses the latest Claude models by default:
- **Claude Opus 4.6** — Most capable, ideal for complex tasks
- **Claude Sonnet 4.6** — Balanced performance and speed
- **Claude Haiku 4.5** — Fastest, ideal for simple tasks
