## What is Claude Code?

Claude Code is Anthropic's official command-line interface (CLI) tool that lets you interact with Claude AI directly in your terminal for various software engineering tasks.

## Core Features

<CardGrid>
  <Card title="Terminal Native" icon="terminal">
    Interact with Claude directly in your terminal, seamlessly integrated into your dev workflow.
  </Card>
  <Card title="Code Understanding" icon="magnifier">
    Automatically understands project structure and code context for precise suggestions.
  </Card>
  <Card title="Tool Integration" icon="puzzle">
    Built-in file editing, search, Git operations and more — all-in-one development assistant.
  </Card>
  <Card title="Extensible" icon="add-document">
    Support for MCP servers, custom skills, and hooks for flexible extensibility.
  </Card>
</CardGrid>
